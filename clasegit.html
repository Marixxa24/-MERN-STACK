Comandos de Git

En este módulo aprenderemos a utilizar Git y GitHub para el control de versiones de nuestro proyecto. Veremos cómo crear un repositorio, hacer commits, push y pull, así como trabajar con ramas y fusionar cambios.

instalacion y creacion del mismo, manejo de los comandos para traer el repo a local y hacer un nuevo archivo el cual vamos a subir con un primer commit y un push al repo de github, para que quede guardado en la nube y podamos compartirlo con otras personas.

Comandos:
  • git init Inicializa un nuevo repositorio de Git en el directorio actual.
  • git add . Agrega todos los archivos del directorio actual al área de preparación para el próximo commit.
  • git commit -m "mensaje" Crea un nuevo commit con el mensaje especificado.
  • git push origin master Envía los cambios del repositorio local al repositorio remoto en la rama master.
  • git pull origin master Obtiene los cambios del repositorio remoto en la rama master y los aplica al repositorio local.
  • git clone url Clona un repositorio de Git desde la URL especificada.
  • git status Muestra el estado actual del repositorio, incluyendo los archivos que han sido modificados o agregados.
  • git log Muestra un registro de todos los commits realizados en el repositorio.
  • git branch Muestra una lista de todas las ramas del repositorio.
  • git checkout -b nombre Crea una nueva rama con el nombre especificado y se cambia a ella.
  • git merge nombre Combina los cambios de la rama especificada con la rama actual.

  • git commit -m "agregando el archivo de git" Crea un nuevo commit con el mensaje especificado.
